## **Minecraft Mod Updater**

Minecraft Mod Updater is a Java-based application designed to simplify and automate the process of updating installed Minecraft mods.

The application checks locally installed mods, identifies them, retrieves version information from the Modrinth platform through its REST API, and determines whether newer compatible versions are available. When an update is found, the application can download the appropriate newer version automatically.

This project was developed as part of a bachelor's thesis focused on software update automation and REST API integration.

## **Features**

Scans locally installed Minecraft mods

Identifies installed mod files

Checks for available updates

Communicates with the Modrinth REST API

Detects newer compatible mod versions

Downloads available updates

Reduces the need for manual mod update management

## **Requirements**

Before building or running the project, make sure the following software is installed:

Java Development Kit (JDK) 11 or newer

Apache Maven
